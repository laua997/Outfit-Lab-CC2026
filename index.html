Your Replicate (optional)

Replicate’s API is not callable directly from GitHub Pages (no CORS). To use your account, deploy the worker in cloudflare-worker/, add the REPLICATE_API_TOKEN secret, then paste the worker URL here. Leave blank to keep using the ITP class proxy.

Use the exact URL Wrangler prints after wrangler deploy (three hostname parts before .workers.dev). Do not paste angle-bracket placeholders from docs or chat.

Why sign in?

On the Spark (free) plan, images are stored as base64 in Realtime Database under your account (no Firebase Storage). Try-on calls go from your browser to the ITP proxy, which turns images into temporary URLs for the model.
